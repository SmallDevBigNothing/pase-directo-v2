## 2024-05-18 - Async Loading States in Vanilla JS
**Learning:** In architectures heavily relying on vanilla JavaScript and inline string interpolation for components (without frameworks like React or complex state management), providing localized loading and success feedback on interactive elements like buttons greatly improves UX without significant refactoring. By dynamically updating `btn.innerHTML` during fetch requests and cooldown periods, users get immediate contextual feedback alongside global toast notifications.
**Action:** When working on similar vanilla JS projects without external templating engines or UI frameworks, prefer simple text-swapping or inline style manipulation on DOM elements for async operations to provide accessible and clear feedback without introducing unnecessary complexity.
